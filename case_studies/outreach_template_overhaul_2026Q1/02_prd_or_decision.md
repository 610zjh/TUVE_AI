# 02. PRD + 2 份 ADR

> PRD-0051 主线 + 2 份 ADR 解决子问题。

---

## PRD-0051 关键段（节选）

### §2 目标
- ✅ 外联回复率从 1.8% 提到 ≥ 2.5%（业界中位数）
- ✅ 转 Demo 率从 11% 回到 ≥ 15%
- ✅ 平均回复时长从 2.4 天降到 ≤ 1.8 天

### §3 用户故事（销售视角）
- "作为销售，我希望发出去的外联能让客户感觉'这不是群发'，所以会愿意回复"
- "作为销售，我希望我每次发完外联后能看到 24 小时内的初步反馈数据"

### §5 验收标准
- [ ] AC-1: 至少 3 个版本的外联模板（按 ADR-0028 的客户分层用）
- [ ] AC-2: 每个模板都过一轮 A/B 测试 + 数据支撑选出最终版本
- [ ] AC-3: 销售能在 CRM 里看到自己的外联回复率（dashboard）
- [ ] AC-4: 客户面文案过红线 #2 三道闸
- [ ] AC-5: 视频素材有 3 个版本（按 ADR-0029 的素材分类）

## ADR-0028: 按客户分层用不同模板

```markdown
# ADR-0028: 按客户分层用不同模板

- 时间：2026-02-08
- 决策人：销售总 - M
- 状态：Accepted

## 背景
单一模板对所有客户已经过时——不同行业、不同规模、不同决策链需要不同语境。

## 选项
- A. 1 套模板（现状）
- B. 5 套（按行业分）
- C. 3 套（按客户决策链：决策者 / 影响者 / 执行者）

## 决策
选 C。理由：
- 业界标准：销售方法论（Challenger Sale 等）按"决策链角色" 分语言
- 长期主义：行业分类会变化（新行业冒出），但决策链角色相对稳定
- 核心目标：精准沟通 = 提升回复率 = 直接服务核心 KPI

## 后果
- 正面：每个客户能感觉"这是写给我的角色的"
- 负面：销售要先判断对方角色，增加 1-2 分钟前置工作
- 复评触发：6 个月后看回复率是否还在 ≥ 2.5%
```

## ADR-0029: 视频素材按 3 类场景分类

```markdown
# ADR-0029: 视频素材按 3 类场景分类

- 时间：2026-02-08
- 决策人：营销总 - K + 视频 lead - V
- 状态：Accepted

## 背景
现有 12 支外联视频缺乏分类，销售用的时候经常"随便挑一支"——和外联模板的关系不清晰。

## 选项
- A. 不分类（现状）
- B. 按行业分 5 类
- C. 按"客户痛点"分 3 类（产能瓶颈 / 质量瓶颈 / 创意瓶颈）

## 决策
选 C。配合 ADR-0028 的"决策链"分层 → 销售可以"对决策者说产能瓶颈视频，对执行者说质量瓶颈视频" 这样匹配。

## 后果
- 正面：销售选视频不再凭感觉
- 负面：每类需要至少 3-4 支视频做素材库——上线后还要持续生产
- 复评触发：每 6 个月看每类视频的实际使用率，使用率 < 10% 的删

## 关联
- ADR-0028（分层逻辑同源）
- PRD-0051
```

---

## 教学点 / Teaching Points

- **数据驱动的目标**："≥ 2.5%（业界中位数）" 比"提升回复率"具体
- **多个 ADR 配合**：分层逻辑 ADR-0028 和素材分类 ADR-0029 互相引用
- **决策援引"业界标准 + 长期主义 + 核心目标"** 三个支柱
- **复评触发条件具体**："6 个月后回复率 < 2.5% 时复评" / "6 个月后视频类使用率 < 10% 时删"
