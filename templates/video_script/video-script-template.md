## 红线自查（拍摄前 + 发布前）/ Red-Line Self-Check

### 字幕 / 台词层面
- [ ] 不出现 PRD 编号 / Bug 编号
- [ ] 不出现内部代号 / 项目代号
- [ ] 不点名其他客户（除非已公开 + 法务过审）
- [ ] 不出现具体合同金额 / 内部价格
- [ ] 不出现"运维" / "我们项目组" 这类内部岗位代号
- [ ] 不承诺未发布的功能（视频在网上比你预期久）

### 画面层面
- [ ] 拍摄时背景里的电脑屏幕 / 白板 / 文档没有内部信息
- [ ] 客户 logo（如出现）有授权
- [ ] 出镜真人有书面授权
- [ ] BGM 已购买商用授权

### 数据 / 承诺层面
- [ ] 数字有依据，能拿出来
- [ ] 不夸大 / 不挑樱桃
- [ ] 比较友商时不点名
- [ ] 法务过审（如涉及承诺 / 比较 / 数据）
