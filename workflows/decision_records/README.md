## 共同纪律 / Shared Discipline

- 红线 #6：非平凡决策必须落字成 ADR
- ADR 编号全公司统一，不分部门
- 旧 ADR 不删，标 Superseded
- 涉及金钱 / 法律 / 合规的 ADR 必须法务过审
- 季度复评所有 ADR，更新状态
