## 共同纪律 / Shared Discipline

无论做哪种分析，都要：
Whichever analysis you're doing:

1. **先明确判断题**——不要"看一下"模式
2. **数据 / 客户名 / 内部信息按红线 #3 脱敏**才能进 AI
3. **每个数字标出处**——抽查关键数字
4. **结论分置信度档**——不要装作 100% 确定
5. **从结论到具体行动**——Top-3 行动 + 负责人 + 截止
