## 共同纪律 / Shared Discipline

- 所有客户面 / 公开内容必须过红线 #2 三道闸（关键词 grep / 品牌声音 / 法务）
- 数字 / 客户案例 / 承诺必须有依据
- 真人出镜 / 客户案例必须有书面授权
- 不依赖 AI 直接"翻译"——本地化必须人重写
