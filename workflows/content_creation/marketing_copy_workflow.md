# 营销文案 / Marketing Copy Workflow

> 适用：营销、运营、销售在写官网文案、广告投放、活动文案、邮件营销时。
> For: marketing / ops / sales writing website copy, ads, event copy, email campaigns.

---

## 一句话 / One Line

**好的营销文案给人**一个具体的行动**和**一个具体的理由**。空话越多，转化越低。
**Good marketing copy gives a specific action with a specific reason.** More fluff = lower conversion.

---

## 4 步法 / 4-Step Process

### 步 1：明确"读者 + 行动 + 理由" / Step 1: Reader + Action + Reason

不是"我要写一份关于 X 产品的文案"，是：
Not "I want copy about product X", but:

> "我要让 [谁] 读完这段后立刻做 [什么具体动作]，因为他能 [得到什么具体好处]。"

```
请帮我把下面三件事写清楚：

读者：[谁，画像 1-2 句]
行动：[他读完后做什么——点击、注册、回复、加微信等具体动作]
理由：[他为什么愿意做——他能解决什么具体问题 / 得到什么具体好处]
```

### 步 2：起 5 个候选 / Step 2: 5 Candidates

```
基于上面的"读者 + 行动 + 理由"，请起 5 个文案候选：

每个候选：
- 标题（≤ 15 字）
- 主体（≤ 50 字）
- CTA 按钮文字（≤ 6 字）

风格要求：
- 候选 1：偏数据 / 量化效果
- 候选 2：偏情境 / 故事代入
- 候选 3：偏对比 / 痛点放大
- 候选 4：偏权威 / 行业地位
- 候选 5：偏简洁 / 直击行动

约束（每个候选都要遵守）：
- 不出现 PRD 编号 / Bug 编号 / 模型 ID / 内部代号（红线 #2）
- 不点名其他客户 / 不点名对手
- 不出现具体合同金额 / 内部价格档位
- 数字必须真实可兑现（如果说"提升 X%"，要有依据）
```

### 步 3：自查 + 微调 / Step 3: Self-Check and Tune

```
对你刚才的 5 个候选，自查：

1. 哪个候选最贴近"读者画像"？
2. 哪个候选最容易引起"我也要做"的冲动？
3. 哪些候选含有"AI 套话"或空洞形容词（高效 / 智能 / 革命性 / 全方位）？把它们点出来。
4. 哪些候选的"理由"是模糊的、客户读完不知道具体能解决什么问题的？

按上面的反思，给每个候选打 0-10 分，并改写最低分那个。
```

### 步 4：法务 + 红线三道闸 / Step 4: Legal + Red-Line Three Gates

营销文案上线前过 3 道闸：
Marketing copy ships only after 3 gates:

| 闸 / Gate | 检查 / Check |
|---|---|
| **关键词 grep 闸** | 扫 `PRD-`, `Bug-`, 内部表名, 模型 ID, 内部岗位代号 |
| **品牌声音闸** | 和 [`templates/customer_brief/voice_guide.md`](../../templates/customer_brief/) 对照 |
| **法务 / 合规闸** | 涉及合同 / 价格 / 风险声明 / 个人信息的，必须法务过 |

## AI 写营销文案的 6 个常见陷阱 / 6 Common AI Marketing-Copy Traps

| 陷阱 / Trap | 表现 / Symptom | 修正 / Fix |
|---|---|---|
| **形容词堆砌** | "高效、智能、便捷、领先、革命性" | 用具体可衡量的描述（"30 秒生成"而不是"高效"）|
| **AI 套话** | "在当今快节奏的时代""值得注意的是" | 直接给出价值主张，删掉过渡 |
| **抽象的客户故事** | "某客户使用后效果显著" | 具体到行业 / 规模 + 量化（"教育-中型客户内容产出从 50/周 升到 500/周"）|
| **承诺超越能力** | "我们 AI 100% 准确" | 改成"AI 辅助 + 人工把控"等不绝对化的措辞 |
| **空泛的 CTA** | "了解更多" | "点击查看 30 秒演示" / "回复 1 预约 Demo" |
| **缺紧迫感（如果合适）** | 没有任何"为什么现在"的理由 | 给一个真实的紧迫理由（早鸟价 / 限时活动 / 季节性需求）|

## 文案的"层级结构" / Copy Hierarchy

任何营销文案的注意力分配：
Any marketing copy's attention budget:

```
标题 / 第一句          → 70% 决定要不要看下去
第一段 / 头 30 字     → 25% 决定要不要看完
余下内容              →  5% 决定要不要点 CTA
CTA 按钮             → 决定要不要行动
```

**反过来想 / Backward**：
- 标题不够好 → 后面再好都没用 / Bad title → rest doesn't matter
- 第一段不够好 → 后面再好读完率会很低 / Bad first paragraph → low completion rate
- CTA 不够清晰 → 全部努力归零 / Vague CTA → all effort wasted

最大注意力分配：标题。**至少花 30% 的写作时间在标题上**。
Largest attention to: title. **Spend ≥ 30% of writing time on the title.**

## 多版本文案的管理 / Managing Multiple Versions

营销活动通常有多个文案版本：
Campaigns usually have multiple versions:

- 短版（朋友圈、社交媒体）/ Short (social)
- 长版（落地页、官网）/ Long (landing page)
- 多语言（中英 / 中粤）/ Multilingual
- 多渠道（抖音、小红书、邮件）/ Multi-channel

不要让 AI"翻译"——专有名词 / 行业黑话 / 文化语境 AI 翻译错率高。
Don't let AI "translate" — high error rate on proper nouns / jargon / cultural context.

正确做法 / Right way:
1. 先确定**核心信息**（"读者 + 行动 + 理由"）
2. 让 AI 基于核心信息为每个渠道**重写**（不是翻译）
3. 由懂目标语言 / 渠道的人审核

## 数据 / 案例的依据 / Backing Up Numbers

营销文案里的"提升 X%"、"客户满意度 Y%"等数字必须：
Numbers like "X% improvement" / "Y% satisfaction" must:

- 出处可查（哪些客户 / 什么时间段 / 什么基线）/ Source available
- 法务过审（如果是承诺性数字）/ Legal-reviewed if promissory
- 不夸大 / 不挑樱桃（如果只取最好客户的数据，要明确说"最好情况"）/ No cherry-picking

未来被客户 / 监管 / 媒体追问时拿得出依据。
So you can produce evidence when questioned.

## 速查 / Cheat Sheet

```
明确：读者 + 行动 + 理由（不是"写一段关于 X 的文案"）

5 候选风格：数据 / 故事 / 痛点 / 权威 / 简洁

3 道闸：关键词 grep / 品牌声音 / 法务

陷阱：形容词堆砌 / AI 套话 / 抽象案例 / 超能力承诺 / 空泛 CTA / 缺紧迫感

层级：标题 70% / 首段 25% / 余下 5%

不要：让 AI 直接"翻译"；只取最佳数据当代表；用空泛 CTA
```
